  OPTION B (PROPER FIX — 10 minutes)
────────────────────────────────────────
Verify a domain in Resend. You already have GoDaddy open — perfect.

If you own a domain (e.g. shelfwise.app, or any domain you have on GoDaddy):

  1) Go to https://resend.com/domains → click "Add Domain"
  2) Enter your domain (e.g. shelfwise.app or mykitchen.xyz)
  3) Resend shows 3 DNS records: SPF (TXT), DKIM (CNAME), MX
  4) Go to GoDaddy → My Products → your domain → DNS → Manage DNS
  5) Add each record exactly as Resend shows them
     • Type: TXT / Name: @ / Value: (Resend's SPF value)
     • Type: CNAME / Name: resend._domainkey / Value: (Resend's DKIM value)
     • (Optional) Type: MX / Name: send / Value: feedback-smtp.resend.com
  6) Wait 5-15 min → refresh Resend → status shows "Verified" ✅
  7) Back to Vercel → add new env var:
       MAIL_FROM=noreply@yourdomain.com
       (or "ShelfWise <noreply@yourdomain.com>" for a nicer sender name)
  8) Redeploy Vercel

Result:
  ✅ Admin notifications work for any address
  ✅ Owner approval emails work for any signup

If you don't have a domain yet — GoDaddy has .xyz for ~$1/year, or you can
use a free subdomain from services like FreeDNS.
