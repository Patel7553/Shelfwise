WHAT'S NEW
──────────

🌗 POLISHED DARK MODE
   Complete rewrite of the dark palette — warmer slate-blue base,
   emerald brand accent, better contrast on text, subtle borders,
   cards visibly float above the page bg. Every screen adapts.

📸 RECEIPT / DELIVERY NOTE SCANNER  ← THE KILLER FEATURE
   Snap a photo of a supplier delivery note, invoice or shop receipt.
   AI (GPT-4o) reads it and extracts every line item with:
     • Product name
     • Quantity + unit
     • Unit cost + total
     • Category + storage type
     • Supplier
   Review the parsed items in an editable table, tick/untick which to
   import, adjust anything, then IMPORT ALL with one tap.
   Location: Dashboard → new fuchsia "🧾 Receipt" button (NEW badge).

💰 COST TRACKING
   • Every product now has a "cost per unit" field (optional)
   • New "Inventory Value" stat card on the dashboard (Σ qty × cost)
   • Currency picker in Settings → Kitchen tab (GBP/USD/EUR/INR/AED/…)
   • Cost carried through the whole flow — receipts, waste log, etc.

⚠️ ALLERGEN TRACKING (UK/EU Natasha's Law compliance)
   • Every product has 14 allergen tick boxes (gluten, nuts, dairy,
     eggs, fish, sesame, sulphites, lupin, molluscs, etc.)
   • Allergen emoji badges show next to the product name in inventory
   • Legal requirement for UK/EU commercial kitchens

📦 REORDER POINTS
   • Set "reorder when qty ≤ X" per product (optional)
   • New "Below Reorder" stat card on the dashboard
   • Small ⚠ REORDER badge on affected products in the inventory list
   • Foundation for supplier auto-ordering (future session)

🏪 SUPPLIER FIELD
   • New "Supplier" field per product (free text for now)
   • Auto-set when items are imported from a receipt

Before deploying the code:

  1. Supabase → SQL Editor → New query
  2. Open supabase/migration-8-cost-allergens.sql in a text editor
  3. Copy the entire file → paste into Supabase
  4. Click Run — you should see "Success. No rows returned"

This adds 5 columns to products and 1 column to kitchens. Safe to
re-run (uses IF NOT EXISTS).
